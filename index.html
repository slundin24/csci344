Sophie's Coursework

Portfolio Template

Designed and built a responsive single-page layout using CSS Flexbox, where each section occupies the full height of the browser window. Leveraged Flexbox properties such as flex-direction, justify-content, and align-items to vertically stack and center content within each section.

Font Switcher

Developed a dynamic font resizing feature using JavaScript to improve accessibility. Created buttons that increase or decrease the font size of key page elements in response to user interaction. The font size is updated in real-time by modifying the style properties of the elements directly in the DOM.

Theme Switcher

Implemented a responsive theme switching feature using JavaScript and CSS. Designed four visual themes—Default, Desert, Ocean, and High Contrast—that dynamically update the page's appearance when a button is clicked. This promotes visual accessibility and user customization.

Taco Temple

Developed a responsive, accessible, and visually engaging restaurant menu layout for "Taco Temple" using HTML, CSS Grid, Flexbox, and Google Fonts. This project replicates a professional restaurant website design with emphasis on semantic structure, responsive layout, clean UI, and accessibility best practices.

PhotoApp Interface

Developed a responsive, component-based social media front-end using React, designed to mirror a real-time user feed. The app fetches data from RESTful API endpoints and renders interactive elements such as posts, user suggestions, and profile data. Emphasis was placed on accessibility, modular design, and seamless user interaction.
